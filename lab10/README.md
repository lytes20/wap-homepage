## Book Library Management

### Built with

- React
- Vite

### State management

- React Context

### Features

- View Books
- Add a book
- Update a book
- Delete a book

### How to run the app locally:

    $ git clone https://github.com/lytes20/wap-homepage.git
    $ cd lab10
    $ npm install
    $ npm run dev
